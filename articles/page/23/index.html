MySQL Stored Procedures - We Like!

Jul 14th, 2006

Just finished reading an excellent but simple article all about MySQL 5’s stored procedures. Never heard of them prior to reading the article, but I must admit that I am mightily impressed by the simplistic, but powerful nature of stored procedures.

A stored procedure is a set of SQL statements that can be stored in the server. Once this has been done, clients don’t need to keep reissuing the individual statements but can refer to the stored procedure instead.

It’s almost like having all your SQL statements stored in a database table, and being able to run them using a simple function call; CALL HelloWorld(); In fact, that is exactly what it is like.

Bit late to use them in Switchboard, but I may use them in some future app.

Ive Stumbled Upon It All!

Jul 4th, 2006

Found a lost article on Techcrunch just a few minutes ago about 12 up and coming web 2.0 startups. A few interesting ones, but one that caught my eye was a community driven site that pushes sites to you based on your interests. StumbleUpon uses a browser toolbar (currently only available for Firefox) to show you new and interesting sites. Simply click the Stumble button and it takes you to a site that should excite you a little.

I love it! I just can’t stop clicking that little button. And most of the sites it shows me have been exactly what I am interested in, so I clicked the “I like it” button on the sites I like, which then added it to my fave pages found using StumbleUpon.

I can then even add this list to my Blog of feed reader using the supplied RSS feeds.

You really gotta admire the simplicity of this model and it just makes me wish that I had thought of it first.
